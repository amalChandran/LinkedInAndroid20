Linkedin authentication using Oauth 2.0

A simple android project implementing Oauth 2.0 for logging in using a LinkedIn account.

View on GitHub Download .zip Download .tar.gz

Getting started:

You need to create a LinkedIn application and get the required API access token and secret. https://developer.linkedin.com/docs/signin-with-linkedin
